Navigation Bar

I'm sure you have already seen my nice navigation bar? The dropdown doesn't link to anywhere but I think it looks pretty cool!

I want to make it so that it highlights the active page. And also to make it 'sticky' like here, which I think would be cool for the homepage where it has a header abover the navbar.

And while we're at it... did you like my link there? Oh, and my strong text? I can also do emphasised text. I can even use special characters to write out the tags that I used... <strong> and <em>.

Impressed yet?

Navigation Pane

So, I probably wouldn't bother with a navigation pane when I already have a navigation bar, but this is just for demonstration! To open the navigation pane, click here (or you can use the icon in the navbar).

Do you like how the rest of the page slides over? Notice that if the page is too narrow it will overlay the navbar instead (otherwise you end up with the rest of the page looking very squished).

EDIT: I have changed it so that the rest of the page doesn't slide over. After changing the structure of the content to support Bootstrap so I could make the Scrollspy I had slightly different margins so it no longer needs the page to slide over.

And the way the links in the pane light up when you hover over them?
